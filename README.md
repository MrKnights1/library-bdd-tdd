# Library BDD/TDD Project

BDD/TDD mini-application for library book borrowing management.

## Setup

```bash
bun install
```

## Running Tests

```bash
# Run unit tests
bun test

# Run BDD tests
bun run test:bdd

# Run all tests
bun run test:all
```

## Project Structure

- `/src` - Source code
- `/tests` - Unit tests
- `/features` - Gherkin feature files
- `/steps` - Cucumber step definitions
